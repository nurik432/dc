# 📲 Инструкция по запуску Payment Userbot

## Что это делает
- Автоматически читает сообщения от DC Next Bot
- Сохраняет каждый платёж
- Присылает тебе уведомление с дневным итогом после каждого зачисления
- Можно запросить статистику командами

---

## Шаг 1 — Получи API_ID и API_HASH

1. Зайди на https://my.telegram.org
2. Войди своим номером телефона
3. → "API development tools" → создай приложение (название любое)
4. Скопируй **App api_id** и **App api_hash**

---

## Шаг 2 — Узнай username бота банка

Открой DC Next Bot в Telegram → нажми на имя → скопируй username (например `dcnextbot`)

---

## Шаг 3 — Получи SESSION_STRING (один раз локально)

На компьютере с Python:

```bash
pip install telethon
python get_session.py
```

Введи API_ID, API_HASH, номер телефона → получишь длинную строку (SESSION_STRING).
**Сохрани её — понадобится на Railway.**

---

## Шаг 4 — Деплой на Railway (бесплатно)

1. Зарегистрируйся на https://railway.app
2. Нажми **New Project → Deploy from GitHub** (залей папку в GitHub)
   - или **New Project → Empty Project → Add Service → GitHub Repo**
3. В настройках проекта → **Variables** → добавь:

| Переменная       | Значение                        |
|------------------|---------------------------------|
| `API_ID`         | твой api_id (число)             |
| `API_HASH`       | твой api_hash                   |
| `PHONE`          | +992xxxxxxxxx                   |
| `SOURCE_BOT`     | username бота банка (dcnextbot) |
| `SESSION_STRING` | строка из get_session.py        |
| `REPORT_BOT`     | me (или твой @username)         |

4. Railway сам установит зависимости и запустит бота.

---

## Команды (пиши себе в "Избранное" / Saved Messages)

| Команда            | Что делает                          |
|--------------------|-------------------------------------|
| `/today`           | Итог за сегодня                     |
| `/all`             | Все дни с суммами                   |
| `/day 05.03.26`    | Конкретный день                     |

---

## Что происходит автоматически

После каждого зачисления бот пришлёт тебе:
```
✅ +18.40 TJS  [09:30]
От: 9920717***0044
📅 За 05.03.26: 40.10 TJS (2 платежей)
```

---

## Вопросы?

Если что-то не работает — проверь:
- SESSION_STRING скопирован полностью (очень длинная строка)
- SOURCE_BOT — точный username бота (без @)
- В Railway логи: вкладка **Deployments → View Logs**
